# Simple Log Analysis with ELK Stack

## Overview
This project demonstrates basic log analysis using the **ELK Stack (Elasticsearch, Logstash, Kibana)**.  
It focuses on ingesting log files, parsing them into structured data, and visualizing insights through Kibana dashboards.

Designed for **beginner SOC analysts (SOC L1)** to gain hands-on SIEM experience.

---

## Tools & Technologies
- Elasticsearch 7.12.1
- Logstash 7.12.1
- Kibana 7.12.1
- Ubuntu 20.04+

---

## What This Project Does
- Ingests log files using Logstash
- Parses logs into searchable fields
- Indexes data in Elasticsearch
- Visualizes login activity, IP addresses, and user actions in Kibana

---

## Key Skills Demonstrated
- SIEM fundamentals
- Log ingestion and parsing
- Elasticsearch indexing
- Kibana dashboards and visualizations
- SOC L1 log analysis concepts
